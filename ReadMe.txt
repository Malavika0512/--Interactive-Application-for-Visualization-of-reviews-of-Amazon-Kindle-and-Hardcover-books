The application runs best on chrome. The following steps are to be followed to execute our project. 
1.	Clone our repository.
2.	Open command prompt and navigate to root folder using cd.
3.	Make sure python3 is installed in the computer
4.	Run this command python3 -m http.server port
5.  Open google chrome and paste "http://localhost:8001/index_bubble.html" where 8001 is the port
